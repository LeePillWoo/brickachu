## ⚠️ 이슈 및 개선사항 (Issues & Feedback)

### 기술적 변경 사항
- **동물 제거 버튼 설계 변경**: 초기 구현에서 🐾 버튼에 스폰/제거 모드 토글을 합쳤으나, UX 혼란으로 분리. 🐾=스폰 전용, 🧹=제거 모드 전용으로 역할을 명확히 나눔.
- **제거 모드에서 블록 조작 비활성화**: 제거 모드(animalMode === 'remove') 진입 시 블록 추가/제거 클릭이 차단됨. 의도된 동작이나, 추후 "제거 모드 중 블록 모드 병행" 옵션 고려 가능.

### 현재 알려진 버그 / 주의사항
- **동물 뿅 효과 중 updateDogs 참조**: `removeAnimalWithEffect` 실행 후 mesh scale 애니메이션 진행 중에 `updateDogs`가 해당 mesh에 접근하지 않도록 animals 배열에서 즉시 제거하는 방식으로 처리됨. 간헐적으로 scale이 baseScale로 리셋될 수 있음.
- **모바일 롱프레스 + contextmenu 충돌**: iOS Safari에서 2초 롱프레스 도중 기본 컨텍스트 메뉴가 뜰 수 있음. `touch-action: none` CSS 또는 `touchstart preventDefault` 추가 필요.
- **먹이 consumeTimer 진행 중 제거 시**: 동물이 이미 먹기 시작한(consumeTimer > 0) 먹이를 클릭 제거하면 foods 배열에서 제거되지만 mesh는 이미 축소 중. 시각적으로 문제없으나 배열 중복 제거 방어 코드 확인 필요.
- **전체 삭제 연기 파티클 성능**: 동물 10마리 동시 삭제 시 100개 SphereGeometry 파티클이 생성됨. 저사양 기기에서 프레임 드롭 가능성 있음. Object pooling 적용 고려.

### 추후 개선 아이디어
- 🧹 버튼 활성 상태일 때 커서를 ✂️ 모양으로 변경 (CSS cursor)
- 제거 모드 진입 시 동물에 빨간 아웃라인 하이라이트 효과
- 개별 제거 시 간단한 "팝!" 사운드 효과 (Web Audio API)
